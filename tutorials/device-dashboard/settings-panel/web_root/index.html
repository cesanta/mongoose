When a field is edited, a local copy is created. For as long as that copy is different from the device's value, the field is added an "edited" CSS class. Bad values (e.g. non-numeric value for the field that is bound to a numeric data) give a "error" CSS class.

The Location field has autosave flag set, so it is saved automatically as the user type text
