LED ${status.led1 ? 'ON' : 'OFF'}
This toggle button is a stylised checkbox. data-bind="status.led1" binds it to the device variable. data-autosave="1" makes it send POST /api/status {"led1": VALUE} to the device every time the user toggles the button. On the device side, sync_status() is called every time the LED state is read or written.
Open this page in two or more browser windows and see how the dashboards stay synchronised when changes are made in any of them.
